# Koota compatibility table

- **koota.gif** – Reference chart: Girl’s Nakshatra (row) × Boy’s Nakshatra (column) → score 0–36.
- **scores.json** – 27×27 grid used by the app. Row = girl nakshatra index (0–26), column = boy nakshatra index (0–26). Edit this file to match the exact numbers in koota.gif if they differ from the default formula.

Nakshatra order (0–26): Ashwini, Bharani, Krittika, Rohini, Mrigasira, Ardra, Punarvasu, Pushya, Ashlesha, Magha, P.Phalguni, U.Phalguni, Hasta, Chitra, Swati, Vishakha, Anuradha, Jyeshta, Mula, P.Ashadha, U.Ashadha, Shravana, Dhanishta, Shatabhisha, P.Bhadra, U.Bhadra, Revati.
